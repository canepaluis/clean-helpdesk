# Sistema de mesa de ayuda

### Descripción
Este sistema permite al usuario una resolución de problemas con el cliente a través de tickets, permitiendo el contacto en tiempo real a través de comentarios.

### Stack
- Backend: Laravel
- Frontend: Vite.js (React)
- Base de datos: MySQL

### Objetivo
Con este sistema se pretende evidenciar el fortalecimiento en las buenas prácticas de estructura y código, además del conocimiento técnico.
